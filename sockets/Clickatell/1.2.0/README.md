# Clickatell Syncano Socket

It is Clickatell integration with Syncano. It allows you to send text messages using the clickatell api

## Endpoints

### send-text-message

#### Parameters:
```

  user: 'companyname',
  password: '******',
  api_id: 'GJD79867',
  from: '2772000000',
  to: '2772000000',
  text: 'Hello world'
```
