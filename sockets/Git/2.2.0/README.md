# Git Syncano Socket

It is Git integration with Syncano. It allows you to machines for working with the `git` command line interface.

## Endpoints

### clone

#### Parameters:

      destination: './put/the/new/local/repo/here',
      remote: 'git://github.com/balderdashy/sails-docs.git',
      branch: 'master'

,
### status

#### Parameters:

      dir: './'

,
### pull-or-clone

#### Parameters:

      destination: './',
      remote: 'origin',
      branch: 'master'

,
### pull

#### Parameters:

      destination: './',
      remote: 'origin',
      branch: 'master'
