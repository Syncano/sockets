# Booleans Syncano Socket

It is Booleans integration with Syncano. It allows you to work with true/false values and boolean operators like and (&&), or (||), not (!), and the ternary qualifier (?/:).

## Endpoints

### to-boolean

#### Parameters:

      string: 'foo'


### is-between

#### Parameters:

      value: 3,
      min: 1,
      max: 4


### is-defined

#### Parameters:

      value: '==='


### is-undefined

#### Parameters:

      value: '==='
