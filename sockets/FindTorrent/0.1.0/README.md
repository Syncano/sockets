# FindTorrent Syncano Socket

It is FindTorrent integration with Syncano. It allows you to find torrents from kickass, eztv, and more

## Endpoints

### query-kickass

#### Parameters:
```

  apiUrl: 'https://kickass.to',
  sortByKey: 'seeders',
  sortOrder: 'desc',
  query: '',
  page: 1,
  category: 'all'
```


### query-eztv

#### Parameters:
```

  baseUrl: 'https://eztv.ag',
  query: '',
  category: 'all'
```


### find-movie

#### Parameters:
```

  query: ''
```

  query: ''
