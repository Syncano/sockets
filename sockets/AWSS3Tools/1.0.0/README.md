# AWSS3Tools Syncano Socket

It is AWSS3Tools integration with Syncano. It allows you to upload files to s3 buckets and more with treeline.

## Endpoints

### upload-to-s3

#### Parameters:

      bucket: 'my_s3_bucket',
      path: 'myfile.jpg',
      acl: 'public-read',
      upload: '/path/to/file',
      secret_key: 'abc1234cged',
      access_key: 'abc1234cge2424ded'
