# KanbanTool Syncano Socket

It is KanbanTool integration with Syncano. It allows you to interact with your kanbantool boards and tasks (http://kanbantool.com/)

## Endpoints

### get-new-cards

#### Parameters:
```

  domain: 'yourcompany',
  apiToken: '11A1AA1AAA1A',
  boardId: 12345,
  fromDate: 12345,
  toDate: 12345
```
