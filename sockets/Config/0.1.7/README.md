# Config Syncano Socket

It is Config integration with Syncano. It allows you to a simple load config (with multiple json, xml or csv file)

## Endpoints

### get

#### Parameters:
```

  path: <*>,
  merge: <boolean>,
  nomethod: <boolean>,
  freeze: <boolean>
```


### multiple

#### Parameters:
```

  schema: <*>,
  freeze: <boolean>
```


### load

#### Parameters:
```

  path: 'test.json'
```
