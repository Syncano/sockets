# Session Syncano Socket

It is Session integration with Syncano. It allows you to read or write data to the session store indicated by the cookie in the current request.

## Endpoints

### load

#### Parameters:

      key: 'loggedInAs'

,
### save

#### Parameters:

      value: 'some string'

,
### del

#### Parameters:
