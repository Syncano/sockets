# Loops Syncano Socket

It is Loops integration with Syncano. It allows you to machines that allow the use of other machines in a loop.

## Endpoints

### map

#### Parameters:
```

  array: <array>,
  worker: <machine>,
  data: '*',
  series: <boolean>
```


### each

#### Parameters:
```

  array: <array>,
  worker: <machine>,
  series: <boolean>
```
